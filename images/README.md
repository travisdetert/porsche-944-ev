# Step-by-step images — gas engine → electric motor

First-pass SVG side-view schematics of the swap. Color key: **red** = gas/removed ·
**green** = kept 944 · **blue** = electric (Leaf) · **orange** = new/HV · **purple** = audio.
These are rough working diagrams — to be refined once the real layout is measured at teardown.

| # | File | Shows |
|---|---|---|
| 1 | `step-1-stock.svg` | Stock: 2.5L gas engine drives the rear transaxle |
| 2 | `step-2-stripped.svg` | Engine, fuel & exhaust out; transaxle + torque tube kept |
| 3 | `step-3-motor-mounted.svg` | EM57 motor + adapter onto the torque tube |
| 4 | `step-4-batteries.svg` | Battery packs mounted low (front + main), ~50/50 |
| 5 | `step-5-complete.svg` | Wired: battery → inverter → motor → wheels, charge port, subs |
