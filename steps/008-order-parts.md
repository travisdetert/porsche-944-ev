# Step 008 — Order parts (batched)

**Phase:** BUY · **Task:** #7 · **Cost:** ~$700–1,200 new
**Blocked by:** 006 (fuse/precharge sized)
**Blocks:** 012

## Do
- [ ] Order the **ZombieVerter VCU**.
- [ ] Order HV bits **not reused** from the donor: DC fuse + precharge (sized in 006), MSD if
      needed, 2/0 cable, lugs, crimper, heat-shrink, orange loom.
- [ ] Order re-source items: electric vacuum pump (brakes), PTC defroster, 12 V + aux audio battery.
- [ ] **Batch it** — the one-at-a-time re-order cycle is the silent schedule killer.

## Done when
Everything new is ordered in one go (reusing donor HV wherever possible).

## Tips & gotchas
- Order the **ZombieVerter early** — ~4-week lead from EVBMW.
- Batch the small HV parts (lugs, cable, fuse, glands) in one order to save shipping.
- Lock the **adapter** quote before this step.
- Head-unit parts can come last — bench it on mock data meanwhile.

## Avoid
- Ordering battery hardware before the bays are measured (step 000).
